# AI 产品尸检馆 - PRD v0.1

## 产品定位
AI 产品尸检馆是一个结构化归档 AI 产品失败 / 停更 / 转型 / 消失案例的研究库。目标不是追热点或消费失败，而是把零散事件整理成可检索、可比较、可复盘的案例资产。

## 一句话定义
一个专门记录、拆解、归档 AI 产品失败 / 停更 / 转型案例的结构化研究库。

## 核心价值
- 理解哪些 AI 产品死了、为什么死
- 提炼死前信号与深层原因
- 为独立开发者、产品经理、创业者提供反面样本
- 形成可筛选、可比较、可持续积累的案例数据库

## 目标用户
1. 独立开发者
2. AI 创业者
3. 产品经理
4. AI 行业观察者
5. 研究者 / 分析师 / 投资人

## 核心需求
- 按死因查案例
- 按赛道查案例
- 按时间查案例
- 快速研究单个失败产品
- 提交线索与补充证据

## MVP 范围
### 要做
- 首页
- 案例列表页
- 案例详情页
- 状态 / 赛道 / 死因筛选
- GitHub 登录（后续接入）
- 提交案例（后续接入）
- 评论 / 收藏（后续接入）
- 基础统计页
- 最小后台与审核机制（后续接入）

### 不做
- AI 产品大全 / 导航站
- 快讯流 / 新闻站
- 开放式 wiki
- 复杂社交系统
- 大规模自动抓取
- 多模型 API 聚合调用

## 产品调性
- 冷静
- 克制
- 结构化
- 有判断
- 有研究价值
- 不幸灾乐祸

## 关键原则
1. 先像工具，再像媒体
2. 先有结构，再有观点
3. 先少而准，再多而杂
4. 先人工保证质量，再考虑自动化
